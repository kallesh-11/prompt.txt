# prompt.txt
2D Graphics Editor in C

Develop a menu-driven 2D Graphics Editor using the C programming language. The program should provide a canvas represented by a two-dimensional character array and allow users to create and manipulate simple graphical objects.

Features to be implemented:
1. Initialize a drawing canvas with blank characters.
2. Display the canvas on the screen.
3. Draw filled rectangles at specified positions.
4. Draw horizontal and vertical lines.
5. Draw triangular shapes with user-defined dimensions.
6. Draw circles using mathematical calculations.
7. Delete previously drawn rectangular regions.
8. Provide a user-friendly menu for selecting drawing operations.
9. Validate coordinates to ensure drawing remains within canvas boundaries.
10. Allow repeated drawing operations until the user chooses to exit.

The project demonstrates the use of arrays, functions, loops, conditional statements, and basic graphics algorithms in C.
